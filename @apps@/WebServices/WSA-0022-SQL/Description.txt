Суть этого сервиса - заполнение двух таблиц XXODI.XXTU_OPERATIONS и XXODI.XXTU_STEPS
данными, которые передаются сервису в параметрах при его вызове.
Таблицы на экземпляре TOPAZ созданы. JNDI = OAADataSource.
В таблицах сейчас содержатся данные по двум технологическим процессам - номера 49 и 4305.
 
Сервис WSA.0022 за один вызов будет получать данные на один процесс.
Сначала нужно удалить из таблиц существующие данные по указанному процессу
 
DELETE FROM xxodi.xxtu_operations
WHERE process_id = 123
 
DELETE FROM xxodi.xxtu_steps
WHERE process_id = 123
