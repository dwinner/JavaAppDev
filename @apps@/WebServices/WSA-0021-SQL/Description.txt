Файл KATTP.DBF содержит список технологических процессов.
Поле NOMTP - номер процесса.
Поле NOM_TIP_TP - номер типового процесса.
Если NOM_TIP_TP не равно 0, значит процесс создан на базе типового а следовательно,
операции процесса - совокупность операций собственных и операций типового процесса.
 
Процесс 14 - процесс типовой.
Процесс 15 - создан на базе типового процесса 14.
Операции процесса 15 - совокупность операций своих (15) и операций типового процесса (14).
Процесс 16 - создан на базе типового процесса 14.
 
Процесс 1141 не имеет типового процесса. Состоит только из собственных операций.
 
Каждый файл *.OPE содержит операции одного из процессов.
Имя файла соответствует номеру процесса.
 
Операции состоят из переходов.
Каждый файл *.PER содержит переходы одного из процессов.
Имя файла соответствует номеру процесса.
Атрибут NOMOP указывает на операцию, к которой принадлежит переход.
